# Dotfiles

Personal development environment managed with GNU Stow.

## Requirements

- macOS
- Homebrew
- Git
- GNU Stow

## Installation

```bash
git clone git@github.com:<your-username>/dotfiles.git ~/.dotfiles
cd ~/.dotfiles
./bootstrap
```

## Structure

```
git/
nvim/
starship/
tmux/
zsh/
```

Each directory is a GNU Stow package.
